# SGJ2024
SGJ2024
## Installation

Create python environment:
```shell
python3.10 -m venv venv
. venv/bin/activate
```

Install the project:
```shell
pip install -e .
```

## Usage
Launch the game:
```shell
launch_game
```
(Optional) Launch in debug mode if you do not have a leapmotion device available:
```shell
launch_game --debug
```
